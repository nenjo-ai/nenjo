# Nenjo Platform

## Purpose
Nenjo is a modular, portable agentic workflow platform that enables teams to define, deploy, and operate sophisticated AI systems using declarative manifests. It combines a hosted platform, Rust SDK, and intelligent bootstrap agent (Nenji) to make building production-grade agentic workflows dramatically faster and more reliable.

## Core Concepts
- Define once, take everywhere (portable manifests)
- Hybrid architecture: convenience + flexibility
- Structured knowledge layer + behavior layer separation
- Self-improving system through intelligent agents

## Key Stakeholders
- Businesses building internal AI systems
- Solo entrepreneurs and consultants
- Developers and AI power users
- Agencies delivering agentic solutions

## Success Criteria
- Agents and routines that are reliable, auditable, and maintainable
- Rapid bootstrapping via Nenji
- Knowledge that is queryable, versioned, and composable
